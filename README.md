# Overview

"Weather Forecast" is a basic weather app developed with Angular & D3.js. Upon submitting your location, the app will display the temperature within a chart.

# Skills Demonstrated:

1. Angular
2. D3.js
3. TypeScript
4. RxJS
5. RESTful API Integration
6. HTML5
7. CSS3
8. Sass
9. Bootstrap
